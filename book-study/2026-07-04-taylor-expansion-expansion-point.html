1. The General Taylor Formula Has Two Points

Known point

The point where we know the function value and derivatives is called a.

a = expansion point

Target point

The point where we want to estimate the function is called z.

z = point to predict
f(z) approx f(a) + f'(a)(z-a) + f''(a)(z-a)^2 / 2! + ...

Key idea: a is not special by itself. It is just the name for the point where we are expanding.

2. Why Gradient Descent Uses x Instead of a

In gradient descent, the current position is normally named x. So we choose:

a = x

This means: expand around the current point.

z = x + epsilon u

This means: predict the function after a tiny step in direction u.

Substitute these into the general formula:

f(x + epsilon u) approx f(x) + f'(x)((x + epsilon u) - x) + ...

Since the difference is:

(x + epsilon u) - x = epsilon u

we get:

f(x + epsilon u) approx f(x) + epsilon f'(x)u + ...

3. Correction From the Photo

The mistaken move

The photo tried to rewrite the expression in a way that effectively expanded around -epsilon u.

not the right center: -epsilon u

The right move

For gradient descent, expand around the current position, because that is where we know the value and derivative.

right center: x

The lesson: Taylor expansion is always centered at the point where the local information is known.

4. The Core Philosophy

Stand at current point
Measure local information
Predict nearby values
Choose a useful step

Taylor expansion is built on this idea:

local information at current point -> prediction near current point

Gradient descent uses the first-order version of this idea. Newton's method uses the second-order version.

5. One-Sentence Memory Hook

Taylor expansion predicts the behavior of a function near the current point using only information measured at that current point.
